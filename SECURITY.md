
# Security Policy

## Supported Versions
El proyecto sigue versionado SemVer y mantiene soporte activo solo en la versión estable principal.

## Reporting a Vulnerability
Si identificas una vulnerabilidad, envía un email privado al equipo técnico del IEB.
No abras un issue público sin aviso previo.

## Vulnerabilities in Dependencies
El repositorio usa Dependabot para detección temprana y actualización de librerías inseguras.
